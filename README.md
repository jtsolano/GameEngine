# GameEngine
Game engine used to study game engine development.

## Objective
The objective of this project is show my C++ skills together with graphics, rendering, gameplay and engine development.


## Project Generation
Project uses premake for generating the project files, placing the sln in the root of the project.


Game Ideas:
- Cell shading
- Procedural Voxel terrain with LOD, octree,
- Skeletal Meshes playing animations and animation blending.
- UI and menus
- Scene Graph, components


Advanced
- GBuffer
- Shadows
